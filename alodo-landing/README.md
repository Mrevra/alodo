# Alodo Platform

Scaffold initial du SaaS Alodo base sur Next.js 16 App Router.

## Structure

- `app/`: routing uniquement
- `src/`: logique metier, services, hooks, types, i18n
- `supabase/`: migrations, seed, policies RLS
- `scripts/`: scripts techniques

## Commandes

```bash
npm run dev
npm run build
```
